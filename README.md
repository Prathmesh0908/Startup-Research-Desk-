# 🚀 Startup Research Desk AI

**Live Demo:** https://startup-research-desk-production.up.railway.app/

Startup Research Desk AI is a full-stack AI-powered startup intelligence platform built to help founders, students, entrepreneurs, incubators, and investors evaluate startup ideas before investing time or money.

This project was designed as an advanced **Agentic AI system** that goes beyond a normal chatbot. Instead of simply generating text, it performs structured startup research, validates business ideas, analyzes competitors, detects risks, and creates investor-style reports.

It simulates the work of a startup analyst, market researcher, and venture capital associate inside one platform.

---

# 📌 Why We Built This

Many people have startup ideas but struggle with questions like:

- Is this idea practical?
- Is the market already crowded?
- Who are the competitors?
- What are the risks?
- Can this generate revenue?
- Would investors be interested?
- How can the idea be improved?

Normally this requires hours or days of research.

Startup Research Desk AI solves this problem in minutes using AI.

# 🛠 Technical Architecture

## Frontend

Built using **Streamlit** to create a responsive SaaS-style web dashboard with:

- modern UI  
- cards  
- tabs  
- metrics  
- progress bars  
- report downloads  

## Backend

Built in **Python** using modular architecture.

Core modules include:

- `app.py` → UI + routing  
- `agents.py` → AI workflows  
- `api_service.py` → rate limiting + retry logic  
- `.env` → API key management  

## AI Layer

Uses **Groq API** with Llama models for:

- reasoning  
- structured responses  
- startup intelligence  
- scoring  
- summarization  

## Search Intelligence

Uses **Tavily API** for real-time web search:

- market trends  
- startup competitors  
- public information  
- industry insights  

## Deployment

Hosted on **Railway** for live public access.
